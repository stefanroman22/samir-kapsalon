# Failure modes the agent caught itself in

Self-observed mistakes — things that went wrong without user intervention. These usually surface from test failures, build errors, or the agent noticing inconsistency mid-build.

## Format

```
## YYYY-MM-DD — <failure mode summary>

**Detected during:** which phase / test / command
**Symptoms:** what went wrong (error message, behavior)
**Root cause:** why it happened
**Fix:** what resolved it
**Prevention:** how to avoid next time (a check, a rule, a habit)
```
